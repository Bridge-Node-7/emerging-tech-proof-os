# Security Policy

## Supported Version

This is v0.1.2 Foundation Skeleton. Security hardening is still in active development.

## Reporting a Vulnerability

Please report security issues through the repository maintainer or GitHub security advisory flow when available.

## Security Posture

This repository includes basic checks for:

- required files
- common secret patterns
- synthetic data safety
- evidence card structure
- documentation quality

These checks are not a substitute for full security review.

For production use, add stronger controls such as secret scanning, dependency review, CodeQL, branch protection, required reviews, and environment-specific security testing.

This repository is designed to contain no real credentials, sensitive data, or restricted operational details.
