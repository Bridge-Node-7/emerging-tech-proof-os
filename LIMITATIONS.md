# Limitations

This is **v0.1.3 Public Proof Boundary Release**.

It is a foundation skeleton, not a complete production system.

## What This Template Can Do

This repository can help a project:

- organize claims
- preserve sources
- show confidence and limitations
- document AI-assisted work
- create review-ready evidence cards
- run baseline repository checks
- make release discipline visible

## What This Template Cannot Do

This repository cannot:

- prove that a claim is true
- replace expert review
- replace legal, security, compliance, or classification review
- guarantee production readiness
- guarantee investment suitability
- prevent every sensitive-data leak
- validate evidence automatically
- protect proprietary content that users choose to publish

## Known Technical Limitations

- Workflows use baseline checks.
- Secret and sensitive-data scans use pattern matching and can miss issues.
- Evidence card checks validate structure, not truth.
- Release packaging creates artifacts, but maintainers must still review them.
- GitHub Actions behavior may need adaptation in copied repositories.
- CODEOWNERS is a starter review-control file and may need owner/team adjustments.

## High-Stakes Use

For high-stakes research, policy, security, safety, funding, infrastructure, national security, or operational use, this template must be supplemented with:

- qualified expert review
- source validation
- legal and IP review
- security review
- production-grade access controls
- branch protection and required reviews
- environment-specific compliance checks

## Public Boundary Limitation

This public repository is not the place for proprietary technology claims, client work, patent strategy, restricted information, or sensitive operational detail.

Public proof should show discipline.

Private work should protect the engine.

## Release Integrity

Release artifacts and checksums should be regenerated after final file edits.

Do not claim final release integrity until the final package has been scanned and the manifest matches the current files.

We are here now. Evidence over hype. Human judgment is final.
